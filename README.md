# Payment Authorization Engine

Sistema Fullstack para simulação de autorização de transações de cartão de crédito.

## Tecnologias

- **Frontend:** React, Redux Toolkit, Bootstrap.
- **Backend:** Node.js, Fastify, TypeScript, MongoDB.

## Como Rodar

### Pré-requisitos

- Node.js
- MongoDB rodando na porta 27017

### Passos

1. Clone o repositório.
2. Backend:
   - `cd backend`
   - `npm install`
   - `npm run dev`
3. Frontend:
   - `cd frontend`
   - `npm install`
   - `npm run dev`
